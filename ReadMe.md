INDIGO

Indigo is a task manager software which is catered for keyboard users. It is a programme that helps its users to organise their daily tasks. It supports natural language input and is a very intuitive product. Some of the basic functions it can do are add, delete, edit, view, undo, and redo. It can also search for tasks and mark them as complete or not complete. It also supports multiple undo, redo, delete, complete and uncomplete fucntions. That means, you can do all those tasks multiple times at one go.

To use the software, please get the jar file provided.
